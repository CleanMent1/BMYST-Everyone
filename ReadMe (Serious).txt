 Tweleve Everyone Mod

	This mod turns every single character/pedestrian in the game into 		Orange Twelve (BMYST) a.k.a "Tweleve" except for CJ, only modify his 		clothes to be the same as Tweleve.

	>> https://github.com/CleanMent1/Tweleve-Everyone-Mod

___________________________________________________________________________

//Needs:

	>> Cleo 4: https://www.mixmods.com.br/2020/10/biblioteca-cleo-4-4/


	>> Modloader: https://www.mixmods.com.br/2018/01/modloader/

___________________________________________________________________________

//Installation:

	Extract the "Tweleve Everyone" folder to the Modloader folder.

//Installation BONUS:

	Extract everything that is INSIDE "BONUS" folder to the "Tweleve 		Everyone" folder.
